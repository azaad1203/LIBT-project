Interaction and Worldwide Web

What is a Graphical user interface ?
  • A business use case
    • Interaction with Human activity system
    • IS (System) Use cases
    • Interaction with an Information System
    • (HCI – Human Computer Interaction)

What is the worldwide web ?
  • A distributed and extensible hypertext
    • References to other documents can "jump" to them.
    • Read in any order.
  • Unregulated distribution of information
  • Each time you click a link the browser "gets" the linked document.
What are Clients and servers ?
  • A client requests a service
  • Servers provide a single response to each request
  • Request-response pairs are independent
What are Urls and web browsers ?
  • Uniform Resource Locator (URL) protocol : //server / "folder" / file # location http://www.en.wikipedia.com/wiki/use_case
  • Process:
    1. Contact server using protocol
    2. Request document
    3. Server locates folder and file
    4. Sends document to browser
    5. Browser scrolls to location
What is the concept of Multi media ?
  • Many extensions for audio, graphics and animation or video.
  • Browsers recognise and handle different formats.
  • Multiple documents on one screen.
    • Embedded graphics
    • Frames
    • Pop-up windows
